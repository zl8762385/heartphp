core::load_config()

配置文件放置路径是：用户项目目录的conf/下.
所有配置文件return变量必须为$config,下面是使用方法:

$config = array();

//系统密钥
$config['server_authkey'] = '!@#$%^&*)()POIUWEQ@';//服务端密钥 后台
$config['domain'] = 'http://heartphp.com/';
//配置静态文件URL
$config['static_path_images'] = 'http://heartphp.com/statics/images/';//images
$config['static_path_js'] = 'http://heartphp.com/statics/js/';//images
$config['static_path_css'] = 'http://heartphp.com/statics/css/';//images
