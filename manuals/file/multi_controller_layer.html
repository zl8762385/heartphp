多层控制器支持

单函数方法获取多层控制器对象。

多层控制器 /libs/controller/读取这下面的文件

//该实例读取路径是 /libs/controller/login/usersController.php读取这下面的文件
//users 文件名称，相当于usersController.php
//login 目录名
$users = C('users', 'login');
$users->test();

//该实例读取路径是 /libs/controller/usersController.php读取这下面的文件
$users = C('users');
$users->test();

分层控制一般用于读取controller目录下的文件，用于控制器之间相互沟通
