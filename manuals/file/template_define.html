模板定义

为了对模板文件更加有效的管理，HeartPHP对模板文件进行目录划分，默认的模板文件定义规则是：

模板目录默认在跟目录tpl/下.

在创建一个控制器是就会指定默认的模版目录，例如：

//创建一个test控制器
//控制器路径 controller/testController.php
class testController extends base_controller {

  public function index(){
    $this->view->display();
  }

  public function test() {
    $this->view->display('test');
  }
}

在tpl/目录下创建对应的模版 例如：
tpl/test/test.html

分层控制一般用于读取controller目录下的文件，用于控制器之间相互沟通
